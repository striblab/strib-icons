Install

Include the stylesheet (in your website <head> or via @import in CSS) to get started.

<link rel="stylesheet" href="{{- .Site.BaseURL -}}strib-icons.min.css" />
@import url("{{- .Site.BaseURL -}}strib-icons.min.css");

To use the SVG sprite instead of the font (i.e. when you want to utilize icons with multiple colors), you can try to utilize the sprite file as described in "Usage" by including the full URL of the sprite file. However, it's possible that may not work because of an issue with chrome. If that is the case, copy the sprite file directly into the codebase to proceed. Sprite file is located at: 

{{- .Site.BaseURL -}}strib-icons.svg
